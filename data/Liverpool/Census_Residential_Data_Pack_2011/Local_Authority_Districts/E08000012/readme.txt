Census Residential Data Pack - 2011 - Local Authority District Geodata Pack: Liverpool (E08000012)


+ Abstract
This geodata pack provides the Census Residential Data Pack for the year 2011 covering the Local Authority District: Liverpool (E08000012)


+ Contents
	 - readme.txt: Information about the CDRC Geodata pack
	 - metadata.xml: Metadata
	 - tables: Folder containing the csv files
	 - shapefiles: Folder containing the shapefiles
