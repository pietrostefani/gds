Access to Healthy Assets & Hazards index (AHAH) - 2019 - Local Authority District Geodata Pack: Liverpool (E08000012)


+ Abstract
This geodata pack provides the CDRC AHAH index for the year 2019 covering the Local Authority District: Liverpool (E08000012)


+ Contents
	 - readme.txt: Information about the CDRC Geodata pack
	 - metadata.xml: Metadata
	 - tables: Folder containing the csv files
	 - shapefiles: Folder containing the shapefiles
